# fade-gallery
Fade Effect Gallery
damo : https://jhs97.github.io/fade-gallery/

## 기능
1. 이전/다음 이미지 보기
2. 인디케이터 버튼 클릭시 이미지 변경
3. 현재 이미지 인디케이터 스타일 업데이트 
4. 자동재생
